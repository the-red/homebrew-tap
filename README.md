# The-red Tap

## How do I install these formulae?

`brew install the-red/tap/<formula>`

Or `brew tap the-red/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "the-red/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
